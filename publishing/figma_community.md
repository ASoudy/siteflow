# Figma Community Listing - FDN Web Intake

## Plugin Name
**FDN Web Intake – Website to Figma**

## Short Description
Convert websites into fully editable Figma designs using real DOM data.

## Full Description
**FDN Web Intake** converts real websites into fully editable Figma designs.

Using structured DOM data exported from the browser, the plugin recreates layout, hierarchy, and styles directly in Figma — without screenshots or manual rebuilding.

FDN Web Intake is part of the **FDN Web** ecosystem and works with **FDN Web Capture**.

### What it does
* Recreates website layouts in Figma
* Preserves hierarchy and positioning
* Supports colors, gradients, borders, shadows
* Accurately renders text styles and spacing
* Produces clean, editable layers

### How to use
1. Export DOM data using **FDN Web Capture**
2. Open **FDN Web Intake** in Figma
3. Paste or upload the JSON file
4. The design is recreated on the current page

### Best for
* UI & UX designers
* Product teams
* Redesign projects
* Design audits
* Documentation workflows

### Important
FDN Web Intake works with **real DOM structure**, not images or AI-generated designs.
