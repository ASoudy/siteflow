## Full Description
**FDN Web Capture** is part of the **FDN Web Intake** pipeline, allowing you to export real website structure and styles for accurate reconstruction in Figma.

The extension reads the live DOM, computes layout and visual styles, and exports a clean JSON format compatible with **FDN Web Intake for Figma**.

### Key Features
* Extracts DOM hierarchy and layout
* Captures spacing, sizes, and positioning
* Preserves colors, borders, shadows, and typography
* Exports structured JSON for Figma import

### How it works
1. Open any website
2. Run **FDN Web Capture**
3. Export DOM data as JSON
4. Import it into **FDN Web Intake** in Figma

### Use cases
* Rebuilding existing websites in Figma
* Redesigning legacy interfaces
* UI documentation and audits
* Design system extraction

### Privacy & Security
* No analytics or tracking
* No personal data collection
* No network requests
* All processing happens locally

## Chrome Permissions Explanation
* **Read and change data on websites**: Required to access DOM structure and computed styles for export
